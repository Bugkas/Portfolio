My Journey with Code

I didn't start coding because of a class; I started because of curiosity. Before I ever sat in a Computer Science lecture, I wrote my first line of code in Python to see how software actually functions. Since then, I have expanded my horizons into Java, C, and Kotlin.

The Passion for Systems

I am particularly drawn to backend development and IoT. One of my proudest moments was building an attendance system application and successfully installing it on my own phone. Seeing a tool I created work in the palm of my hand solidified my desire to create profitable, real-world applications. I love the "Aha!" moment when a complex logic bug is finally solved through persistence.

Technical Expertise & Tools

Developing robust systems requires a deep understanding of data flow and hardware integration. I focus on creating scalable architectures that can handle real-time data processing.
