My Journey with Code

I didn't start coding because of a class; I started because of curiosity. Before I ever sat in a Computer Science lecture, I wrote my first line of code in Python to see how software actually functions. Since then, I have expanded my horizons into Java, C, and Kotlin. Currently, I am mastering VB.NET and HTML to round out my full-stack capabilities.

The Passion for Systems

I am particularly drawn to backend development and IoT. One of my proudest moments was building an attendance system application and successfully installing it on my own phone. Seeing a tool I created work in the palm of my hand solidified my desire to create profitable, real-world applications. I love the "Aha!" moment when a complex logic bug is finally solved through persistence.

My Learning Timeline

  1. Self-Taught Beginnings: Exploring Python and basic logic structures out of pure interest.
  2. Academic Entry: Beginning formal Computer Science studies and mastering Python, Java and C fundamentals.
  3. Mobile Development: Architecting and deploying a functional attendance system app.
  4. Hardware Exploration: Diving into IoT projects to see how code influences the physical world.
  5. Web Integration: Learning HTML, CSS, and VB.NET to build comprehensive business solutions.
